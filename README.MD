# 支付宝小程序组件库

## 安装
```bash
npm install --save miniapp-alipay-selene-ui
```

## 引入
在package.json中声明
```bash
{
  "usingComponents": {
    "preview-image-list": "miniapp-alipay-selene-ui/preview-image-list/index"
  }
}
```

## 组建API

### preview-image-list  图片预览组件

| 参数名称 | 描述 | 类型 | 是否必填 | 默认值 |
| ---- | ---- | ---- | ---- | ---- | ---- |
| 单元格 | 单元格 | 单元格 | 单元格 | 单元格 | 单元格 |
| 单元格 | 单元格 | 单元格 | 单元格 | 单元格 | 单元格 |


#### 示例
```bash
<preview-image-list />
```
